About This Project

This project is a simple yet functional To-Do List application, built using HTML, CSS, and JavaScript.

Features

  • Add and manage tasks efficiently
  • Set due times for tasks
  • Interactive user interface

    • Technologies Used

  • HTML5 for structuring the content
  • CSS3 for styling and animations
  • JavaScript for interactivity

    • Future Enhancements

  • Persistence using Local Storage
  • Improved UI/UX
  • More features like task categories, notifications, etc.
